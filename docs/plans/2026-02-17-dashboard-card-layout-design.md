# 仪表盘卡片布局设计方案

## 概述

将基金和股票的列表显示改为流式卡片布局，解决列表显示不全的问题，同时保持所有现有功能。

## 用户需求

- 桌面端和移动端动态适应
- 使用流式布局
- 详细版卡片信息
- 操作按钮始终显示

## 设计方案

### 1. 整体布局
- 左右分栏保持不变（基金在左，股票在右）
- 各自使用流式卡片网格布局

### 2. 响应式断点
| 断点 | 屏幕宽度 | 列数 |
|------|---------|------|
| Mobile | < 768px | 1 列 |
| Tablet | ≥ 768px | 2 列 |
| Desktop | ≥ 1280px | 2 列 |
| Large Desktop | ≥ 1600px | 3 列 |

### 3. 基金卡片设计

```
┌────────────────────────────────────────┐
│ 💰 基金名称                      320007 │
├────────────────────────────────────────┤
│ 持有份额          成本价        当前净值 │
│ 1,000.00         1.5000       1.6523  │
├────────────────────────────────────────┤
│ 持仓市值                          收益率 │
│ ¥1,652.30                       +10.15% │
├────────────────────────────────────────┤
│ [📜 记录] [➕ 加减仓] [🗑 删除]       │
└────────────────────────────────────────┘
```

### 4. 股票卡片设计

```
┌────────────────────────────────────────┐
│ 📈 股票名称                      000001 │
├────────────────────────────────────────┤
│ 持股数          买入均价      当前股价   │
│ 500             10.50        12.30     │
├────────────────────────────────────────┤
│ 日涨跌          日收益        持仓总额   │
│ +2.50%         +125.00      6,150.00  │
├────────────────────────────────────────┤
│ 总收益                              +16.19% │
├────────────────────────────────────────┤
│ [📜 记录] [➕ 加减仓] [🗑 删除]       │
└────────────────────────────────────────┘
```

### 5. 视觉规范

- 卡片背景: `bg-gradient-to-br from-[#0d0d15] to-[#12121a]`
- 边框: `border border-[#2a2a3a]`，悬浮时变为对应主题色
- 基金主题色: 青色 `#00ffff`
- 股票主题色: 橙色 `#ff9500`
- 按钮始终可见，固定在卡片底部

### 6. 功能保持

- ✅ 搜索添加基金/股票
- ✅ 拖拽排序（卡片可拖动）
- ✅ 加减仓操作
- ✅ 查看交易记录
- ✅ 删除持仓

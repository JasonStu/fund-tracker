# 首页基金模块添加估算涨幅字段

## 需求概述

在首页基金卡片（FundCard）的"收益率"旁边添加"估算涨幅"字段，显示基金的实时估算涨跌幅，方便用户在不进入详情页的情况下快速查看基金的当日涨跌情况。

## 设计方案

### 实现方案

在 FundCard 组件的数据网格中，将"收益率"从 `col-span-2` 改为 `col-span-1`，然后在旁边新增一个"估算涨幅"字段。

### 数据源

- `position.estimatedChangePercent` - 估算涨跌幅百分比

### 显示逻辑

- 涨（≥0）：红色 `#ff3333`
- 跌（<0）：绿色 `#33ff33`
- 格式：`+X.XX%` 或 `-X.XX%`

### 布局调整

```
当前布局 (grid-cols-3):
| 持有份额 | 成本价 | 当前净值 |
| 持仓市值 | 收益率(col-span-2) |  |

调整后:
| 持有份额 | 成本价 | 当前净值 |
| 持仓市值 | 收益率 | 估算涨幅 |
```

## 技术细节

### 修改文件

- `src/app/(dashboard)/page.tsx` - FundCard 组件

### 无需修改

- 类型定义（`estimatedChangePercent` 字段已存在）
- API 路由（数据已存在）
- 详情页（保持不变）

## 验收标准

1. 基金卡片显示"估算涨幅"字段
2. 涨幅颜色正确（涨红色，跌绿色）
3. 格式正确（带正负号和百分号）
4. 布局整齐，与其他字段对齐
